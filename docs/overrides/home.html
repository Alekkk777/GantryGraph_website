from gantrygraph import GantryEngine, gantry_tool
from gantrygraph.perception import DesktopScreen
from gantrygraph.actions import MouseKeyboardTools
from langchain_anthropic import ChatAnthropic

@gantry_tool
async def notify_slack(channel: str, msg: str) -> str:
    """Post a message to a Slack channel."""
    return await slack.post(channel, msg)

agent = GantryEngine(
    llm=ChatAnthropic(model="claude-sonnet-4-6"),
    perception=DesktopScreen(),
    tools=[MouseKeyboardTools(), notify_slack],
    max_steps=30,
)
agent.run("File the quarterly report and ping #finance")

Architecture

Simple loop. No magic.

Four deterministic steps. Every decision is observable and logged.

01 / OBSERVE
See the world

Take a screenshot, render the DOM, or read any custom sensor. The agent always knows what it's looking at.

02 / THINK
Decide what to do

The LLM receives the observation and decides which tool to call next — or declares the task complete.

03 / ACT
Execute with guardrails

Tool calls run through the approval gate. Errors are fed back automatically so the agent self-corrects.

04 / REVIEW
Loop or finish

No more tool calls? Task complete. Otherwise loop back — up to max_steps.
